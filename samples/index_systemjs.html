SAMPLE TS + SystemJS

Compilato con: 
		tsc js/app.ts --module system --target es5 --allowJs --outFile bundle_system.js
Equivale a: tsc -p . 
// tsconfig.json
{
    "compilerOptions": {
        "module": "system",
        "target": "es5",
        "noImplicitAny": false,
        "sourceMap": false,
        "lib": [ "dom", "es2015" ],
		"outFile": "bundle_system.js",
		"allowJs": true
    },
	"files": [
		"js/app.ts"   
    ]
}
